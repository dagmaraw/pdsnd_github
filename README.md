### Project Title
pdsnd_github

### Description
This project provides a user with data on bikeshare usage in Chicago, New York City, and Washington.
The user chooses a city and can filter by month and day of the week.

### Files used
* bikeshare.py
* chicago.csv
* new_york_city.csv
* washington.csv
